# 🍅 FocusSprout

**FocusSprout** is a sleek and modern Pomodoro web app designed to help you stay focused and productive. Built with cutting-edge tools and a beautiful UI, it's a perfect companion for time-blocking your work and breaks.

---

## 🔥 Features

- ⏱ Pomodoro timer with customizable focus and break sessions
- 🌓 Theme Selection Supported
- 📈  Spotify Playlist integration
- 📱 Fully responsive and mobile-friendly
- ⚡ Fast performance powered by Vite
- 🔔 Toast notifications and smooth animations

---

## 🛠 Tech Stack

### **Frontend**
- **React** – Component-based UI
- **TypeScript** – Type-safe development
- **Vite** – Lightning-fast build tool
- **Tailwind CSS** – Utility-first styling
- **ShadCN UI** – Prebuilt modern UI components

### **Form & Validation**
- **react-hook-form** – Performant forms
- **Zod** – Schema-based validation

### **Routing & State**
- **react-router-dom** – Declarative routing
- **@tanstack/react-query** – Data fetching and caching

### **UI/UX Enhancements**
- **Framer Motion** – Animation library
- **lucide-react** – Icon pack
- **sonner** – Toast notifications
